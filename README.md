# Hotkeys

# Build instructions
```sh
Install Git and Nodejs
```
```sh
git clone https://github.com/m2rk1312/shitterClient
```
```sh
cd shitterClient
```
```sh
npm i
```
If you just want to run the program from console/terminal:
```sh
npm start
```
If you want to build it:
```sh
npm run dist
```
